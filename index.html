Hello World!

There is a set of basic requirements common for all projects at Microverse that will help you avoid common mistakes. Please take a while to read them now: Use descriptive names for your branches and make sure that your feature branch is up to date with the main branch in your repository. Use meaningful names for your Pull Requests and add a summary of its content as described here. Use descriptive commit messages as described here. Add a descriptive README file to your repositories as described here. Do not commit files that are not related to the project. Instead, add them to .gitignore file.
